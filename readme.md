# My Personal Website

## Tech

- [Zola](https://www.getzola.org/)
- [Turbo](https://turbo.hotwire.dev/)
- [minireset.css](https://jgthms.com/minireset.css/)
- HTML + SCSS + JS

## Running

1. Install [Zola](https://www.getzola.org/).
2. Clone the repo.
4. ```sh
   zola serve
   ```
