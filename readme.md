# My Personal Website

## Tech

- [Zola](https://www.getzola.org/)
- [Turbolinks](https://github.com/turbolinks/turbolinks)
- [minireset.css](https://jgthms.com/minireset.css/)
- HTML + SCSS + JS

## Running

1. Install [Zola](https://www.getzola.org/) and [Node.js](https://nodejs.org/).
2. Clone the repo.
4. ```sh
   zola serve
   ```
